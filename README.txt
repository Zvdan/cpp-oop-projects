# C++ OOP Projects

This repository contains a collection of console-based C++ projects developed to practice Object-Oriented Programming concepts.

## Projects

### Student Management System
Manage students and their enrolled courses.

### Library Management System
Manage books, customers, borrowing, and returns.

### Bank Management System
Create accounts, deposit, withdraw, transfer funds, and verify users through PINs.

### Hospital Management System
Manage patients, doctors, and appointments.

### Quiz Application
Create quizzes, add questions and answers, and calculate scores.

## OOP Concepts Practiced

- Classes and Objects
- Encapsulation
- Controller Pattern
- Separation of Concerns
- Vectors
- Header and Source File Organization
- Menu-Driven Console Applications

## Language

- C++
