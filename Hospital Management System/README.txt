# Hospital Management System

A C++ console application for managing patients, doctors, and appointments.

## Features

- Add Patient
- Remove Patient
- Add Doctor
- Assign Patient to Doctor
- Cancel Appointment
- View All Patients
- View All Doctors
- View Doctor's Patients
- Auto-generated IDs

## Project Structure

- Patient
- Doctor
- HospitalController
- HospitalUI
- main

## Concepts Used

- Classes and Objects
- Encapsulation
- Vectors
- Controller Pattern
- Menu-Driven Application

## Notes

- Patient IDs are generated automatically.
- Doctor IDs are generated automatically.
- Doctors maintain lists of assigned patients.
- Removing a patient automatically removes associated appointments.
