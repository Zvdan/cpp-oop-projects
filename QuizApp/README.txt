# Quiz Application

A C++ console application for creating and taking quizzes using Object-Oriented Programming principles.

## Features

- Add Questions
- Add Multiple Answers
- Specify Correct Answer
- Take Quiz
- Calculate Score
- Display Final Result

## Project Structure

- Question
- QuizController
- QuizUI
- main

## Concepts Used

- Object-Oriented Programming
- Classes and Objects
- Encapsulation
- Vectors
- Controller Pattern
- Menu-Driven Application

## Notes

- Questions can contain multiple answer choices.
- Each question has one correct answer.
- The application calculates the final score automatically.
- The controller handles quiz logic while the UI handles user interaction.
