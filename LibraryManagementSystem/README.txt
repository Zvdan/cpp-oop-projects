# Library Management System

A C++ console application for managing books and borrowing operations in a library.

## Features

- Add Book
- Add Customer
- Borrow Book
- Return Book
- View Available Books
- View Borrowed Books
- Auto-generated Book IDs
- Auto-generated Customer IDs

## Project Structure

- Book
- Customer
- LibraryController
- LibraryUI
- main

## Concepts Used

- Object-Oriented Programming
- Classes and Objects
- Encapsulation
- Vectors
- Controller Pattern
- Menu-Driven Programming

## Notes

- Books can only be borrowed if available.
- Customers maintain a list of borrowed books.
- IDs are generated automatically by the system.
