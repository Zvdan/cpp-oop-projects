# Bank Management System

A C++ console application that simulates basic banking operations.

## Features

- Create Account
- Deposit Money
- Withdraw Money
- Transfer Money Between Accounts
- Display Account Information
- PIN Verification
- Auto-generated Account IDs

## Project Structure

- Account
- BankController
- BankUI
- main

## Concepts Used

- Object-Oriented Programming
- Encapsulation
- Controller Pattern
- Vectors
- Pointer Usage
- Menu-Driven Design

## Notes

- Each account has a unique ID generated automatically.
- Accounts are protected using a PIN.
- Accounts are created with an initial balance.
- The system displays the updated balance after deposits and withdrawals.
