# Student Management System

A C++ console application built using Object-Oriented Programming principles to manage students and their courses.

## Features

- Add Student
- Delete Student
- Add Course to Student
- Delete Course from Student
- Display All Students
- Auto-generated Student IDs

## Project Structure

- Student
- StudentController
- StudentUI
- main

## Concepts Used

- Classes and Objects
- Encapsulation
- Vectors
- Header Files (.h)
- Source Files (.cpp)
- Controller Pattern
- Menu-Driven Console Application

## Notes

- Student IDs are generated automatically by the system.
- Courses are managed using course lists associated with each student.
- Course deletion is performed using numbered selections to avoid typing mistakes.
